# Usage

From the repo root directory use this command on terminal:
``` 
$ ghci Src/Main.hs 
``` 
Once inside of haskell console call main function:
```
*Main> main
Bem vindo ao SAD!

Kilian

*Main>
``` 
You should get this output.
